Hackathon Squad - MWIS Optimizer
A highly optimized, multi-start C++ approximation engine for the NP-Hard Maximum Weight Independent Set (MWIS) problem.

- Algorithm: Implements the FR14 Forest + Dynamic Programming algorithm.
- Performance: Designed to rigorously execute millions of random permutations to find near-optimal nodes within a strict 5-minute execution window.
- Stack: C++17, Fast I/O, mt19937 Random Number Generation.
